## ðŸ¡ Teleportation & Homes
Teleporting helps you navigate the world safely. Note that some teleports have a short warmup timer.

| Command | Description |
| :--- | :--- |
| `/spawn` | Return to the main server spawn. |
| `/sethome [name]` | Save your current location (Max: **3 homes**). |
| `/home [name]` | Teleport to one of your saved homes. |
| `/tprandom` | Teleport to a random location in the wilderness. |
| `/tpa [player]` | Request to teleport to another player. |
| `/tphere [player]` | Request a player to teleport to you. |
| `/tpaccept` | Accept an incoming teleport request. |
| `/tpdeny` | Decline an incoming teleport request. |

## âš”ï¸ Combat & PvP
Choose how you want to play. Remember: the Arena is always dangerous!

| Command | Description |
| :--- | :--- |
| `/pvp on` | Enable PvP for yourself (Wilderness only). |
| `/pvp off` | Disable PvP for yourself (Wilderness only). |

> [!IMPORTANT]  
> PvP is **forced ON** inside the Arena near spawn. You cannot toggle your PvP status while inside the arena boundaries.

## ðŸ›¡ï¸ Land Protection
We use **GriefPrevention** to keep your builds safe. To start, find or craft a **Golden Shovel**.

> [!TIP]  
> **How to claim:** Right-click one corner of your house with a Golden Shovel, then right-click the opposite corner. The area in between will be protected!

| Command | Description |
| :--- | :--- |
| `/trust [player]` | Give a friend permission to build/break in your claim. |
| `/containertrust [player]` | Allow a player to open chests/furnaces only. |
| `/accesstrust [player]` | Allow a player to use buttons and levers only. |
| `/untrust [player]` | Remove all permissions for a player. |
| `/claimslist` | View your active claims and remaining claim blocks. |
| `/abandonclaim` | Delete the claim you are currently standing in. |
